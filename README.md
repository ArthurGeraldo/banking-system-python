# Banking System in Python

Projeto simples de **sistema bancário** desenvolvido em Python, executado no terminal.  
Permite criar um usuário, realizar depósitos, saques e consultar saldo, com persistência de dados em arquivo JSON.

## Funcionalidades
- Cadastro de usuário
- Depósito de valores
- Saque com verificação de saldo
- Consulta de saldo
- Salvamento automático dos dados em JSON

## Observação importante
O arquivo `conta.json` **é criado automaticamente** quando o usuário é cadastrado e realiza o **primeiro depósito**.  
Não é necessário criar o arquivo manualmente.

## Tecnologias usadas
- Python 3
- Programação Orientada a Objetos
- JSON

## Como executar
1. Clone o repositório
2. Abra a pasta do projeto
3. Execute o arquivo principal:
